## Data Engineer Bootcamp (VDT-C01) – 8 Tuần (16 Buổi)

Khóa học **Data Engineer Bootcamp** được thiết kế để trang bị cho học viên các kỹ năng cần thiết để trở thành Data Engineer chuyên nghiệp. Khóa học bao gồm 3 giai đoạn: Engineering Foundations, Data Engineering Core, và Advanced & Production — giúp bạn xây dựng nền tảng vững chắc và sẵn sàng cho môi trường thực tế.

### Đối tượng phù hợp

- ✔ Sinh viên có nền tảng lập trình cơ bản
- ✔ Engineer muốn chuyển ngành sang Data
- ✔ Data Analyst muốn nâng cấp sang Data Engineering

### Tech Stack

Khóa học sẽ trang bị cho bạn các công nghệ hàng đầu trong ngành Data Engineering:

- **Linux, Bash, Docker, Docker Compose** — Nền tảng & Dev Environment
- **Python, FastAPI, Pydantic, pytest** — Lập trình & API
- **PostgreSQL, SQL, SQLAlchemy, DBeaver** — Database & truy vấn dữ liệu
- **Apache Kafka, Debezium** — Ingestion & Change Data Capture (CDC)
- **MinIO, Delta Lake, Trino, Hive Metastore** — Data Lakehouse architecture
- **Apache Spark, PySpark** — Batch processing
- **Apache Flink, Flink SQL** — Stream processing
- **ClickHouse, Apache Pinot** — OLAP engines
- **dbt** — Analytics Engineering
- **Apache Airflow** — Orchestration & scheduling
- **Great Expectations, Soda** — Data Quality
- **DataHub** — Metadata management & Lineage
- **Feast** — Feature Store
- **Terraform, GitHub Actions** — Infrastructure as Code & CI/CD
- **Prometheus, Grafana** — Monitoring & Alerting

### Tổng quan theo Phase

| Phase                                  | Buổi      | Chủ đề                                                                  | Output                                  |
| -------------------------------------- | --------- | ----------------------------------------------------------------------- | --------------------------------------- |
| **Phase 1: Engineering Foundations**   | B1 – B5   | Linux, Python, SQL, FastAPI, Docker                                     | Dev environment + API + ETL pipeline    |
| **Phase 2: Data Engineering Core**     | B6 – B12  | Kafka, CDC, Lakehouse, Spark, Flink, OLAP, Airflow, Data Quality       | End-to-end data pipeline                |
| **Phase 3: Advanced & Production**     | B13 – B16 | dbt, Terraform, CI/CD, Capstone Project                                 | Production-ready project + Demo Day     |

### Syllabus chi tiết theo buổi (16 buổi)

#### PHASE 1 — Engineering Foundations (Buổi 1–5)

**Buổi 1 — Linux, Shell & Dev Environment**

_Lý thuyết:_

- GNU/Linux core commands, Bash scripting
- Makefile, Cron, SSH, tmux

_Lab:_

- ✔ Cài Ubuntu trên WSL2/EC2; viết CLI crawler thu thập dữ liệu text từ web

_Tech stack: Ubuntu, Bash, Makefile, AWS EC2_

**Buổi 2 — Python cho Data Engineering**

_Lý thuyết:_

- Syntax, data structures, error handling, argparse
- File I/O, context managers
- venv / uv; async vs sync

_Lab:_

- ✔ Setup môi trường ảo; viết ETL script đơn giản xử lý CSV và JSON

_Tech stack: Python, uv virtualenv_

**Buổi 3 — Database & SQL**

_Lý thuyết:_

- Relational vs NoSQL concepts
- DDL/DML/DQL/DCL/TCL
- Indexing, query plan, EXPLAIN
- Transactions & ACID

_Lab:_

- ✔ Tạo schema PostgreSQL, tối ưu câu query, dùng SQLAlchemy ORM

_Tech stack: PostgreSQL, SQLAlchemy, DBeaver_

**Buổi 4 — Web API & FastAPI**

_Lý thuyết:_

- REST vs GraphQL
- Auth (JWT, API key, OAuth2)
- Request validation, API healthcheck
- Performance testing (Locust)

_Lab:_

- ✔ Xây dựng API serving dữ liệu với auth; chạy load test Locust

_Tech stack: FastAPI, Pydantic, Locust_

**Buổi 5 — Testing, Docker & Docker Compose**

_Lý thuyết:_

- TDD, pytest, unittest
- Docker architecture, images, volumes, network
- Docker Compose, private registry

_Lab:_

- ✔ Containerize FastAPI app; viết test suite; deploy local stack với Compose

_Tech stack: Docker, Docker Compose, pytest_

#### PHASE 2 — Data Engineering Core (Buổi 6–12)

**Buổi 6 — Ingestion Layer: Kafka & CDC**

_Lý thuyết:_

- Source systems (files, OLAP/OLTP, CDC, logs)
- Kafka internals (producer/consumer, partitions, offsets, KRaft)
- Debezium CDC

_Lab:_

- ✔ End-to-end CDC pipeline: PostgreSQL → Debezium → Kafka topic

_Tech stack: Apache Kafka, Debezium, PostgreSQL_

**Buổi 7 — Storage Layer: Lakehouse Architecture**

_Lý thuyết:_

- Data Warehouse vs Data Lake vs Data Swamp
- Lakehouse pattern, multi-hop (Bronze/Silver/Gold)
- Table formats (Delta Lake, Iceberg, Hudi)

_Lab:_

- ✔ Xây Lakehouse từ đầu: MinIO (S3-compatible) + Delta Lake + Hive Metastore + Trino

_Tech stack: Delta Lake, MinIO, Trino, Hive_

**Buổi 8 — Batch Processing: Spark (Phần 1)**

_Lý thuyết:_

- Spark architecture & execution model
- RDD vs DataFrame
- Catalyst Optimizer, Project Tungsten, serialization

_Lab:_

- ✔ Viết Spark batch job xử lý dữ liệu Bronze → Silver (cleansing, dedup, schema enforcement)

_Tech stack: Apache Spark, PySpark, Delta Lake_

**Buổi 9 — Batch Processing: Spark (Phần 2)**

_Lý thuyết:_

- AQE, join strategies (broadcast, shuffle sort merge, bucket join)
- Data skew handling
- Spark UI phân tích, dynamic allocation, compaction

_Lab:_

- ✔ Tối ưu pipeline Silver → Gold: xử lý skew, tune memory, compact small files

_Tech stack: Apache Spark, PySpark, Spark UI_

**Buổi 10 — Stream Processing: Flink**

_Lý thuyết:_

- Streaming concepts & notions of time
- Flink DataStream API, Table API, Flink SQL
- Window strategies, watermarks, late data
- State & fault tolerance, backpressure

_Lab:_

- ✔ Real-time pipeline: Kafka → Flink stateful aggregation → sink vào Delta Lake

_Tech stack: Apache Flink, Flink SQL, Kafka_

**Buổi 11 — Consumption Layer: DWH, OLAP & Feature Store**

_Lý thuyết:_

- Dimensional modeling, SCD Type 1/2/3
- OLAP engines (ClickHouse, Apache Pinot)
- Materialized views
- Feature Store concepts & ML integration

_Lab:_

- ✔ Xây dimensional model, tạo materialized views; implement feature pipeline với Feast

_Tech stack: ClickHouse, Apache Pinot, Feast, dbt_

**Buổi 12 — Orchestration, Lineage & Data Quality**

_Lý thuyết:_

- Airflow architecture, operators, XCom, backfill, idempotency
- Dynamic DAGs, task concurrency
- Data validation (Great Expectations, Soda)
- Circuit breaker, data contracts
- Lineage với DataHub

_Lab:_

- ✔ Deploy Airflow trên Docker; viết multi-step DAG với data quality checks và lineage tracking

_Tech stack: Apache Airflow, Great Expectations, DataHub_

#### PHASE 3 — Advanced & Production (Buổi 13–16)

**Buổi 13 — Analytics Engineering & dbt**

_Lý thuyết:_

- Analytics Engineering role
- dbt project structure, models, tests, docs
- Incremental models, snapshots (SCD)
- Deployment (dbt Cloud / self-hosted), CI integration

_Lab:_

- ✔ Build dbt project trên top của Gold layer; viết tests và tạo documentation site

_Tech stack: dbt, DuckDB, GitHub Actions_

**Buổi 14 — Infrastructure as Code & DataOps**

_Lý thuyết:_

- Terraform cho data infra (S3, MSK, EMR, Redshift, MWAA)
- CI/CD với GitHub Actions
- Docker image build pipeline
- Secrets management
- Monitoring & alerting (Prometheus, Grafana)

_Lab:_

- ✔ Provision toàn bộ data stack trên AWS bằng Terraform; setup CI/CD pipeline tự động test và deploy

_Tech stack: Terraform, GitHub Actions, Prometheus, Grafana_

**Buổi 15 — End-to-End Capstone Project (Phần 1)**

_Lý thuyết:_

- Thiết kế kiến trúc hệ thống dữ liệu end-to-end
- Requirements analysis
- Infra provisioning
- Ingestion setup (Kafka + CDC)
- Lakehouse setup (S3 + Delta Lake)

_Lab:_

- ✔ Project thực tế (e-commerce / fintech dataset) — ingestion layer + storage layer hoàn chỉnh

_Tech stack: Kafka, Delta Lake, Terraform_

**Buổi 16 — End-to-End Capstone Project (Phần 2) + Demo Day**

_Lý thuyết:_

- Transformation pipelines (Spark batch + Flink stream)
- Consumption layer (ClickHouse + dbt)
- Dashboard, data quality, lineage
- Performance tuning

_Lab:_

- ✔ Hoàn thiện toàn bộ pipeline, present architecture, Q&A và career roadmap

_Tech stack: Spark, Flink, dbt, ClickHouse, DataHub, Superset, Airflow_
